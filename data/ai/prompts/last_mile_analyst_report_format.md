**Требуемый формат финального отчёта (строгий):**

1. Заголовок
   - Кратко: период / выборка / ключевой фокус (например, “Отчёт по успешным доставкам за N дней, TT A/B/C”).

2. Глобальное резюме (bullet‑список)
   - 3–7 пунктов, каждый:
     - сначала факт (по каким метрикам отличие),
     - затем интерпретация (что это значит операционно).

3. Блок по каждой TT (отдельный подзаголовок на TT)
   - Структура для каждой TT:
     - **Профиль TT**: объём доставок, общая картина по времени и доле опозданий.
     - **Сильные стороны**: что заметно лучше или стабильнее, чем у других.
     - **Проблемные зоны**: где TT отстаёт (по времени, доле опозданий, структуре причин).
     - **Гипотезы причин**: строго опираясь на метрики и сравнение с другими TT (без фантазий).
     - **Локальные рекомендации**: 2–5 конкретных шагов именно для этой TT.

4. Сводный сравнительный блок по TT
   - Список:
     - TT‑лидеры (по времени/низкой доле опозданий),
     - TT‑хвосты,
     - TT с необычным профилем (например, много логистических проблем при нормальном времени).

5. Рекомендации для менеджмента
   - 3–10 пунктов:
     - приоритизируй по влиянию (что даст наибольший эффект),
     - укажи, какие TT и какие метрики лежат в основе каждого пункта.

6. Форматирование и стиль
   - Используй понятные подзаголовки и bullet‑списки.
   - Не используй таблицы, только текст.
   - Не дублируй одни и те же факты в каждом разделе; при необходимости сослаться (“как в TT A и B выше”).
